**Advanced Google Map Block** is a flexible Gutenberg block that lets you add customizable and responsive Google Maps without requiring an API key.

== Description ==

**Advanced Google Map Block** is a powerful yet user-friendly WordPress plugin designed for adding Google Maps directly in the Gutenberg editor. This plugin provides a seamless way to insert location maps with advanced customization options and responsive designs, perfect for business sites, portfolios, or any site needing a map view.

**Key Features:**
- **No API Key Needed:** Add Google Maps effortlessly without worrying about API configurations.
- **Responsive Design Options:** Customize responsive settings to make maps look great on any device.
- **Extensive Styling Options:** Modify map colors, zoom levels, controls, and more for a tailored user experience.
- **Multiple Map Styles:** Choose from different pre-set themes (e.g., dark mode, light mode) or create your own.
- **Dynamic Location Marker:** Easily add markers with custom descriptions or info boxes.

Whether you’re showcasing a store location or highlighting specific landmarks, **Advanced Google Map Block** offers an intuitive way to make maps that match your site’s design and needs.

== Installation ==

1. **Upload & Install:** Upload the `AdvancedGoogleMapBlock` folder to the `/wp-content/plugins/` directory, or install it directly through the WordPress plugins screen.
2. **Activate:** Go to the 'Plugins' screen in WordPress and activate **Advanced Google Map Block**.
3. **Get Started:** Navigate to the Gutenberg editor, insert the **Advanced Google Map Block**, and start configuring your custom map.

== Frequently Asked Questions ==

= Do I need a Google API Key? =

No, **Advanced Google Map Block** does not require an API key to function, making it easy to set up and use.

= Can I add multiple markers on the map? =

Yes, you can add as many markers as needed, each with customizable information pop-ups.

= Is the map responsive? =

Yes, the map is fully responsive and has options to customize the view on different devices.

= Can I style the map to match my theme? =

Absolutely! The plugin includes extensive styling options for colors, controls, and layout to fit any design.
